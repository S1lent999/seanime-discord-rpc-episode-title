# Discord RPC Episode Title for Seanime

A Seanime extension that displays anime episode titles in Discord Rich Presence with a clean SXXEXX format.

## Features
✅ Shows episode title in Discord RPC (e.g., "Driven into a Corner! Broken Zangetsu")

✅ Displays episode code in SXXEXX format (e.g., S01E38)

✅ Uses anime title as the activity name ("Watching Bleach")

✅ Fallback to simple episode number if title is unavailable

## Requirements

- Seanime **3.9.1-alpha.7 or higher**

## Installation

1. Go to Seanime → Settings → Extensions → Add Extensions
2. Click "Install from URL"
3. Paste this link: `https://raw.githubusercontent.com/S1lent/seanime-discord-rpc-episode-title/main/discord-rpc-episode-title.json`
4. Enable the extension

## Notes

- Uses the official Seanime plugin system
- No external API calls (uses internal Seanime data)
- Lightweight and fast
